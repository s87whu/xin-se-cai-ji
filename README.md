# 心色彩记 (xin-se-cai-ji)

一款基于 Flutter 开发的情绪日记应用，帮助用户记录每日心情与日记内容，通过情绪分析了解自身心理状态。

## 项目简介

心色彩记是一款专注于情绪记录的日记应用，支持六种情绪状态（开心、平静、一般、焦虑、难过、愤怒），用户可以：

- 记录每日日记并选择对应情绪
- 查看情绪统计与分析图表
- 使用 AI 助手进行情绪疏导
- 自定义主题与字体大小
- 本地存储，保护隐私

## 技术栈

- Flutter 3.29.0+
- Dart 3.8.0+
- GetX 状态管理
- Isar 本地数据库
- Material Design 3

## 小组成员与分工

| 成员 | 角色 | 负责模块 |
|------|------|----------|
| 成员A | 组长 | 基础架构与日记核心功能 |
| 成员B | 组员 | 情绪分析与图表展示 |
| 成员C | 组员 | AI 助手与设置页面 |
| 成员D | 组员 | 锁屏功能与用户协议 |

## 项目结构

```
lib/

```
